# Breakout Game (Java)

This project is a simple implementation of the classic Breakout game developed in Java.

## Description
The game allows the player to control a paddle to bounce a ball and break bricks. The objective is to clear all bricks without letting the ball fall below the paddle.

## Requirements
- Java Development Kit (JDK)
- Java-compatible operating system

## How to Run
1. Compile the program:
```bash
javac Breakout.java
